## Naming Rules

1. **Format**: `{descriptor}-icon.{ext}`
2. **Case**: kebab-case (lowercase with hyphens)
3. **Suffix**: Always end with `-icon` for icon files
4. **Logos**: Use `-logo` suffix instead of `-icon`
5. **Descriptive**: Use clear, descriptive names
6. **Singular**: Use singular form (e.g., `submission` not `submissions`)
7. **No abbreviations**: Write full words (e.g., `research` not `reserch`)
